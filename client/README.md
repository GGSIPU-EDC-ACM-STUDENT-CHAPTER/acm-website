The official website for the **GGSIPU East Delhi Campus ACM Student Chapter**. This platform serves as a hub for our events, technical blogs, projects, and team updates, empowering students to innovate and collaborate in the field of computing.

## üíª Getting Started

### Prerequisites

Ensure you have Node.js installed (v18 or higher recommended).

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Arsh199965/ACM-Website.git
    cd ACM-Website/client
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

To create an optimized static build:

```bash
npm run build
```

This will generate an `out` folder containing the static assets, ready for deployment.
